I'm Malik Rameez — a graphic designer by intuition, a full-stack developer by obsession, and a storyteller by default. I build brands that look as sharp as the code that powers them, and I ship web experiences that feel less like websites and more like interactive films.

For the last 2 years I've been blending type, motion, and architecture into work for startups, studios, and the occasional Fortune 500. My toolkit spans Figma, After Effects, Node, React, and whatever new AI model decided to drop this week. I like clean grids, buttery animations, and clients who say "make it cooler."

When I'm not pushing pixels or squashing bugs, I'm usually prototyping a side project, reading about typography history, or convincing my cat that the keyboard is not a bed. If you've got a vision that needs both an artist's eye and an engineer's brain — let's build it.
